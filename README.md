# Sidewalk Safari

Sidewalk Safari is a native SwiftUI iOS app that turns a short family walk into a local sidewalk quest. It includes starter quests, editable clue tiles, find moments, a persistent Safari Log, privacy-first copy, and a premium theme preview with StoreKit-unavailable recovery.


## v1.1 Route Kits + Walk Recap

The next version adds quick Route Kits for School Walk, Bus Stop, Park Gate, and Errand Loop. A parent can create an editable custom quest from a kit, copy any quest for today’s sidewalk, and review a local Walk Recap with completed clues, skipped clues, Find Moments, replays, and the next suggested action.
